# Campaign Images — /kamuskay4sga page

Place all photos for the Kamuskay & Makayla joint ticket page in this folder.

## Recommended file names

### Makayla Hughes slider photos
- `makayla-campus.jpg`       — Makayla on Berea's campus
- `makayla-bsu.jpg`          — Makayla at the Black Student Union / Black Cultural Center
- `makayla-bwi.jpg`          — Black Women Initiative event
- `makayla-cheer.jpg`        — Makayla on the Berea College cheer team

### Ticket / joint photos
- `ticket-hero.jpg`          — Hero background or side image (Kamuskay + Makayla together)
- `kamuskay-profile.jpg`     — Kamuskay headshot / profile photo for candidate card
- `makayla-profile.jpg`      — Makayla headshot / profile photo for candidate card

## Usage
Reference images in the page as: `/images/campaign/<filename>`
e.g. `src="/images/campaign/makayla-campus.jpg"`

## Guidelines
- Prefer JPEG for photos, PNG for graphics/logos with transparency
- Recommended minimum resolution: 800 × 600 px
- Recommended aspect ratio for slider slides: 4:3
- Recommended aspect ratio for profile cards: 3:4 or 1:1 (square)
